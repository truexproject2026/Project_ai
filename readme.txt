เป้าหมาย: วิเคราะห์ความรู้สึกลูกค้าและร่างคำตอบอัตโนมัติด้วย AI

------------------------------------------------------------
[คำเตือนและข้อควรระวังก่อนเริ่มใช้งาน]
------------------------------------------------------------
1. API KEY สำคัญมาก:
   - โปรเจกต์นี้ใช้ Groq API (Llama 3.3-70b)
   - ต้องตั้งค่าไฟล์ .env ในโฟลเดอร์หลัก โดยใส่บรรทัดนี้:
     GROQ_API_KEY=ใส่_API_KEY_ของคุณ_ที่นี่

2. ข้อจำกัดของ Groq API (Free Tier):
   - ส่งคำถามได้ไม่เกิน 30 ครั้งต่อนาที (RPM)
   - ใช้งานรวมไม่เกิน 1,000 ครั้งต่อวัน (RPD)
   - หากขึ้น Error 429 ให้รอ 1 นาทีแล้วลองใหม่

3. ความถูกต้องของ AI:
   - AI ร่างคำตอบตามข้อมูลเมนูและแบรนด์ที่กำหนดในโฟลเดอร์ /data
   - ผู้ใช้ "ต้อง" ตรวจสอบคำตอบก่อนกดยืนยัน (Approve) เสมอ

------------------------------------------------------------
[ขั้นตอนการรันโปรเจกต์]
------------------------------------------------------------
1. ติดตั้ง Library (ทำครั้งแรกครั้งเดียว):
   เปิด Terminal แล้วพิมพ์:
   npm install

2. เริ่มรันโปรเจกต์ (Development Mode):
   เปิด Terminal แล้วพิมพ์:
   npm run dev

3. เข้าใช้งานผ่าน Browser:
   เปิดไปที่ http://localhost:3000

------------------------------------------------------------
[โครงสร้างไฟล์ที่สำคัญ]
------------------------------------------------------------
- /app/api/reply/route.ts : ระบบหลังบ้านที่เชื่อมต่อกับ Groq AI
- /app/components/ResultCard.tsx : หน้าจอแสดงผลและการวัดผล AI
- /data : ข้อมูลเมนูร้านและตัวอย่างการตอบ (ใช้ทำ RAG-lite)
- /reports : ไฟล์สรุปการวัดผล (Metrics) ตามเกณฑ์โปรเจกต์
